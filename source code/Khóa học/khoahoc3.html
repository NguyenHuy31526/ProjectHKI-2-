[Giới thiệu CSS]

CSSS là một kỹ thuật dùng để định dạng các thẻ trong trang web

CSS là gì?

CSS là viết tắt của Cascading Style Sheets
CSS mô tả cách các phần tử HTML sẽ hiển thị trên màn hinh, trang web hoặc các phương tiện khác
CSS giúp định dạng trang web rất nhanh nhờ nhiều kiểu định dạng tag, class, element,...

Lợi ích của CSS

  • Dễ quản lý, bao trì

  • Tái sử dụng. Một quy luật kiểu dáng có thể áp dụng cho nhiều thành phần web khác nhau

  • Cải thiện tốc độ

    • Thẻ Style

    Là một tập hợp các đặc điểm nhận dạng cho thành các thành phần trong trang

    Để định dạng, ta chuyển sang chế độ code, rồi định nghĩa các stlye bên trong thẻ <style>

    Sau đây là mã nguồn của một số màu sắc giúp bạn trang trí

    Nhúng CSS vào trang web

    Có 3 cách để nhúng mã CSS vào trong một tài liệu HTML:

  • Khai báo trực tiếp ở thẻ HTML (Inline stle)

  • Khai báo thẻ <style> trong phần thẻ <head> của phần HTML (Internal style)

  • Khai báo trong file .css riêng biệt (External style)

    • Video học HTML5 từ A đến Z

    Mỗi giây phút học tập hôm nay là một khoản đầu tư cho sự thành công của bạn trong tương lai.
