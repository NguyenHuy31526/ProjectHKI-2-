JavaScript là gì?

JavaScript hay còn được viết tắt JS
JavaScript là ngôn ngữ lập trình chạy trên browser, nhằm tạo khả năng tương tác với người dùng.
Mỗi lệnh được kết thúc bằng dấu " ; "
Có thể viết mã lệnh JavaScript trong thẻ <script> của trang, hoặc viết riêng trong file script.js rồi nhúng vào trang html, hoặc viết trực tiếp mã lệnh JavaScript trong phần sự kiện của các thẻ html

Sử dụng mã JavaScript vào trang

  • Viết trực tiếp trong trang với thẻ <script>

  • Viết mã JavaScript trong file riêng

    • 1. Tạo file JavaScript

  • Nhấp menu File -> New -> JavaScript -> Create

  • Gõ mã lệnh JavaScript

  • Lưu file với tên mở rộng là .js, ví dụ: chao.js

    • Liên kết file js đến trang html

  • Mở một file html

  • Nếu vị trí muốn chèn (thường trong thẻ <style>) rồi vào gỡ lệnh chèn

    • Để nắm đầy đủ nội dung bạn có thể tham khảo ở dưới đay
